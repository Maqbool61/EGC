# Support Policy

## Supported Versions

| Version | Status | Security Updates |
|---------|--------|-----------------|
| 0.9.x (latest) | Active | Yes |
| 0.8.x and earlier | End of Life | No |

Only the latest stable release receives security updates. Users are encouraged to upgrade to the latest version promptly.

## Security Update Policy

Security vulnerabilities are patched in the current stable release only. When a vulnerability is confirmed:

1. A fix is developed and reviewed privately
2. A new patch release is published
3. The security advisory is disclosed after the fix is available

End-of-life versions do not receive security backports. If you are running an unsupported version, upgrade to the latest release to receive security fixes.

## Support Scope

Community support is provided through [GitHub Issues](https://github.com/Fmarzochi/everything-gemini/issues).

For security vulnerabilities, use the private reporting channel described in [SECURITY.md](SECURITY.md).

Feature requests and bug reports are welcome via GitHub Issues. Response times are best-effort for community contributions.
